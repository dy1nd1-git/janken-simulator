# ジャンケン・シミュレーター (Python)

このプロジェクトは、異なる戦略を持つプレイヤー同士のジャンケンをシミュレーションし、その結果を統計的に出力するプログラムです。

## 概要

2人のプレイヤー（のび太・スネ夫）が指定回数の対戦を行い、それぞれの勝率を表示します。また、ランダム性が仕様通りに機能しているかを検証するテストコードも含まれています。

### プレイヤーの戦略

- **のび太 (N):** グー・チョキ・パーを均等にランダム（各1/3）に選択します。
- **スネ夫 (S):** これまでの勝負で最も勝率（勝利数）が良い手を選びます。最高勝率の手が複数ある場合は、その中からランダムに選択します。

2. **バージョンの確認**
   ターミナルで以下のコマンドを実行し、バージョンが表示されれば準備完了です。

   ```bash
   python3 --version
   ```

1. シミュレーション実行:

```bash

```

2. のび太のランダム性テスト:

```bash
    python3 test.py
```

## 仕様

- のび太: 常にランダム
- スネ夫: 過去の勝率が最も高い手を選択
- 判定: アイコは1回と数えるが勝敗には含めない
